    Pyodide

    The new scryptIQ written materials are written in our new parchmentIQ template, with learning materials running Pyodide in the background. Pyodide is a very useful software package that allows us to run live Python code cells from inside the webpage itself; thus, the code you see inside the materials has actually run, live. Pyodide also allows us to generate editable code cells, in which you can edit the Python code, and immediately try out what you've learned, without having to leave the web page.

    Types of Cells

    Code cells that have a green output box below are fixed and can not be user-edited.

    However, those with a pink output box below and a pink Edit button, can have their code altered by the user and can be run independently,each time you hit the pink Run Code button. Data stored in variables from previously run cells, both fixed and editable, should carry forward into subsequent editable cells.

    REMEMBER

    If you refresh the webpage in your browser, this will restart and erase Pyodide's memory, which will overwrite any previously executed actions, restoring the page to its previous state.

    There will also be Practice Exercises scattered throughout the materials. These questions should be quick problems to solve, focusing on the subject material you are reading. Answering these earnestly should help progress your understanding much more quickly.

    Pyodide Issues

    Rarely, a code cell may not render with Pyodide. To rectify this, just refresh the page.

    If this or any other issues occur, please contact us at admin@learntodiscover.ai or report the problem via our course feedback form.
