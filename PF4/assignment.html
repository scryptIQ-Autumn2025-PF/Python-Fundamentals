    Assignment PF4

    Assignment Background:

    This short assignment contains three sections of questions that build upon and incorporate methods, techniques and concepts taught and demonstrated in the lesson materials. Each section will explore your understanding of strings, lists, tuples, and sets.

    All questions aim to consider biological context. Timely submission will result in your assignment being returned within 14 days of the submission deadline. Marked Jupyter Notebook manuscripts will be returned with comments from your marker If you receive a pass or distinction, your marker will attach the model solutions for this assignment via the open pull request on your lesson's GitHub repository.

    Assignment Questions:

    Question 1

    Write a Python script that reads the file Data/cell_features.txt and converts it into a nested dictionary with the following structure:
    Steps:

    1. Use proper file handling using the with statement

    2. Read in every line in the file

    3. Use a for loop to access all lines in turn

    4. First, print the data from the for loop to see the structure

    5. Use the information from step 4 to separate the prospective values from the keys

    6. Each new set of cellular features starts after every 5 lines

    7. Use this to ensure the data is inserted into the dictionary at the right point

    8. The outer dictionary keys should be the cell IDs

    9. Each inner dictionary should contain the four feature measurements
    {
'LYMPH_001': {                  # Line 1 (string)
    'DIAMETER': 12.5,           # Line 2 (float)
    'NUCLEUS_RATIO': 0.42,      # Line 3 (float)
    'MITOCHONDRIA_COUNT': 450,  # Line 4 (int)
    'VIABILITY': 0.95           # Line 5 (float)
},
'NEURO_002': {
    'DIAMETER': 18.3,
    'NUCLEUS_RATIO': 0.38,
    'MITOCHONDRIA_COUNT': 820,
    'VIABILITY': 0.88
},
# ... etc for all cells
}

    How to give the answer:
    Show your answer by looping over the first two items in the parent dictionary, printing both
    • Use break with enumerate() to prematurely terminate the loop after two iterations
    Hints:
    • Check out the FASTQ example for how to handle repetitive patterns

    • Remember to remove newline characters (\n) at the beginning

    • Remember to typecast the numerical strings to the right numerical type

    Question 2

    Analyse Cell Features by Type

    Write a function called analyse_cell_type() that performs some statistical analysis on the cell data dictionary you created in Question 1. The aim of the function is to find the mean of each feature per cell type - "LYMPH" and "NEURO".

    We will give you the function docstring to help guide you:

    """
Calculate average measurements for a specific cell type.

:param cell_data: All data containing multiple cell types and their measurements
:type cell_data: dict
:param cell_type: Cell type prefix to analyze (e.g., 'LYMPH' or 'NEURO')
:type cell_type: str
:return: Dictionary with average values for each measurement
:rtype: dict
"""


    Function requirements:
    1. Loop over the dictionary

    2. Check if the cell ID matches the cell type supplied as an argument to the function

    3. If it matches, have each measurement add to a variable to get the total

    4. Count the number of cells for the given cell type

    5. Find the average for each measurement, round the answer to 2 decimal places with round()

    6. Store the averages in a dictionary called results which is returned
    How to give the answer:
    • Call the function twice, once with "LYMPH" and once with "NEURO", and store the returned variable appropriately

    • Print both variables to compare the answer
